# Snakio - Python Pygame Snake Game

Snakio is a classic Snake game implemented in Python using the Pygame library. This project is designed for learning game development concepts and practicing Python programming.

## Features
- Smooth snake movement and growth
- Randomly appearing food
- Score and high score tracking
- Game over detection on collision with walls or self
- Fun background music and sound effects
- Welcome screen and interactive controls

## Controls
- Arrow keys: Move the snake
- Space: Start game or increase score (bonus)
- Enter: Restart after game over

## Technologies Used
- Python 3
- Pygame library
- File handling for storing high scores

## How to Run
1. Install Pygame: `pip install pygame`
2. Download the repository and navigate to the project folder
3. Run the game: `python snake_game.py`

Have fun playing and customizing your own Snake game!
